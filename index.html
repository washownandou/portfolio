📌 使用須知

🔄 股價更新機制（智慧快取）
  • 為節省流量，系統會自動判斷各市場「最近一次收盤時間」，已抓過且收盤價未更新的股票會跳過。
  • 台股交易時間：週一至週五 09:00 ~ 13:30（台北時間），13:30 收盤後當日股價定案。
  • 美股交易時間：週一至週五 22:30 ~ 翌日 05:00（台北時間，含日光節約調整），收盤後當日股價定案。
  • 盤中按更新會抓即時最新價；收盤後再按只會抓尚未取得當日收盤價的股票。
  • 想強制重抓所有股票（忽略快取），請點右上角的 ⚡ 強制更新 按鈕。
  • 修改持股股數或成本均價時，僅做即時計算，不會觸發 API 抓取，可放心調整。
💰 開倉盈虧計算規則
  • 開倉盈虧只計入「有填成本均價」的股票，未填成本的股票完全不會納入計算。
  • 所以您可以只追蹤想觀察盈虧的股票（填成本），其他股票留空成本欄即可，不會被誇大盈虧。
  • 總淨值則包含所有有持股的股票（不論有沒有填成本）。
  • 獲利率% 同樣只看有填成本的股票，公式：(現價 - 成本) ÷ 成本 × 100%。
📈 資產走勢圖（重要！避免誤解）
  • 折線圖會在每次按「🔄 更新股價」成功後自動記錄當日總淨值與開倉盈虧，同一天多次更新會以最後一次為準。
  • ⚠️ 不會回填歷史資料：圖表只能呈現「您開始使用後」的紀錄，無法顯示加入此功能之前的資產走勢。
  • ⚠️ 不代表真實獲利曲線：折線只反映「目前持股 × 當天股價」的計算結果，不會考慮歷史買賣紀錄。例如您今天加入了新持股，圖表會看起來淨值「突然暴增」，但這不是您「賺到了」，而是新增了部位。
  • ⚠️ 修改持股 / 成本 / 刪除列：會直接影響後續所有計算，但過去已記錄的圖表點不會回溯修改。
  • ⚠️ 幣別影響：紀錄當下使用的顯示幣別（TWD/USD）會直接存入，若中途切換幣別，舊資料仍是當時記錄的幣別。建議長期固定用同一種幣別觀察。
  • ⚠️ 圖表雙軸範圍不同：左軸（總淨值）和右軸（開倉盈虧）的數值範圍通常不同，圖中兩條線的「相對位置」沒有絕對意義，請看各自軸的刻度比較。
  • 沒有持股的日子不會被記錄；有持股但成本未填仍會記錄「總淨值」(開倉盈虧該日會顯示 0)。
  • 非交易日（週末、假日）若按更新，當天也會被記錄，因此圖表線可能出現「假日水平延伸」的視覺現象。
  • 所有歷史紀錄存在您本機瀏覽器中，換瀏覽器/換裝置/清快取會全部消失，建議定期匯出 Excel 備份。
💾 資料儲存
  • 所有資料儲存在「您目前的瀏覽器」中（localStorage），換瀏覽器/換裝置/清快取會消失。
  • 想跨裝置使用 → 點 🔗 分享連結 把連結存入書籤，下次打開即可還原（不含歷史紀錄）。
  • 想備份歷史紀錄 → 點 📊 匯出 Excel，存下檔案。
  • 分享連結含您的全部持股資料（股數、成本），請勿貼到公開場合。
📈 資料來源與限制
  • 股價資料來自 Yahoo Finance，非即時報價，可能有 15~20 分鐘延遲，僅供參考，不構成任何投資建議。
  • 台股代號格式：4~6 位數字（如 2330、0050、00713），支援 ETF 含字母結尾（如 00981A）。
  • 美股代號格式：1~5 位英文字母（如 AAPL、TSLA、MU）。
  • 不支援：興櫃 / 未上市 / 加密貨幣 / 期權期貨 / 港股 / 日股。
  • USD/TWD 匯率取自 open.er-api.com，每次按更新會自動刷新。
💡 使用技巧
  • 手機/平板使用 → 善用 ⚙️ 欄位設定 隱藏不需要的欄位，畫面更精簡。
  • 拖曳每列最左邊的 ⋮⋮ 可調整投資組合的顯示順序。
  • 拖曳欄位標題可調整欄位左右順序。
  • 成本均價未填寫時，開倉盈虧與獲利率欄位會顯示「-」，且不會被計入總開倉盈虧。
  • 可切換顯示幣別（TWD/USD），所有金額會自動換算。
  • 想長期追蹤 → 建議每天收盤後按一次「更新股價」，圖表才有完整紀錄。

⚙️ 欄位設定

拖曳調整順序，勾選顯示／取消勾選隱藏。「拖曳手把」「市場」「股票代號」「刪除」為固定欄位。
